# 🚀 AI-Native Project: Building at Inference Speed

## 👋 Introduction

This project explores a new paradigm of software development, where code is no longer written line by line, but **emerges from autonomous systems operating at the speed of AI inference**.

It combines three cutting-edge principles that are currently gaining traction on X and in the AI engineering community.

---

## 🧠 Core Principles

### 1. AutoResearch (Autonomous Experimentation)

Inspired by **AutoResearch by Andrej Karpathy**, this project adopts an **autonomous research loop** where AI agents continuously improve the system through experimentation.

**Key idea:**

* Code is not manually optimized
* Instead, agents:

  * Propose changes
  * Execute experiments
  * Evaluate results
  * Iterate continuously

**Loop:**

```text
propose → run → measure → select → repeat
```

This transforms development into a **self-improving system driven by feedback loops**.

---

### 2. Attention Is All You Need (Transformer Architecture)

Based on the seminal paper:

> *Attention Is All You Need*
> Ashish Vaswani, Noam Shazeer, Niki Parmar, Jakob Uszkoreit, Llion Jones,
> Aidan N. Gomez, Łukasz Kaiser, Illia Polosukhin

This principle provides the **foundation for modern large language models (LLMs)** used in this project.

**Key idea:**

* Replace recurrence and convolution with **attention mechanisms**
* Enable:

  * Parallel computation
  * Long-range dependency modeling
  * Scalable architectures

---

### 3. Shipping at Inference-Speed (AI-Native Development)

Inspired by **Peter Steinberger’s article “Shipping at Inference-Speed”**, this project embraces a new engineering philosophy:

> Software should be built as fast as models can generate and validate it.

**Key ideas:**

* Developers act as **orchestrators**, not coders
* The bottleneck shifts from typing speed → **model inference speed**
* Development loop becomes:

```text
prompt → generate → run → verify → refine
```

This enables **real-time iteration and rapid shipping cycles**.

---

## ⚙️ System Design Philosophy

This project is built around the concept of:

> **Agent-driven software development**

Where:

* AI agents handle:

  * Code generation
  * Execution
  * Testing
  * Optimization
* Humans define:

  * Goals
  * Constraints
  * Evaluation criteria

---

## 🔁 Unified Loop

At its core, the system merges all three principles into a single loop:

```text
objective → prompt → generate → execute → evaluate → improve → repeat
```

* **AutoResearch** → drives iteration
* **Transformers** → power intelligence
* **Inference-speed** → defines velocity
